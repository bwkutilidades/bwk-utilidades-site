# BWK Utilidades - Monorepo

This repository contains the full stack application for BWK Utilidades.

## Structure

- **`/storefront`**: The customer-facing e-commerce frontend (React/Vite).
- **`/admin`**: (Future) Admin dashboard.
- **`/backend`**: (Future) MedusaJS backend.
- **`/docs`**: Project documentation.

## Getting Started

To work on the storefront:

```bash
cd storefront
npm install
npm run dev
```

See `/docs` for more operational details.
